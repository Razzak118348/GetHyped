# Gethyped 🚀

A high-performance, visually stunning creative agency landing page built with **React 19**, **Vite**, and advanced animation libraries. This project showcases a "motion-first" design philosophy, featuring smooth scrolling, interactive mouse trails, and complex GSAP orchestrations.

![Project Status](https://img.shields.io/badge/Status-Live-brightgreen)
![Tech Stack](https://img.shields.io/badge/Stack-React_|_GSAP_|_Tailwind-blue)

🔗 **Live Demo:** [gethyped121.netlify.app](https://gethyped121.netlify.app/)

---

## ✨ Key Features

- **Advanced Animations:** Implementation of GSAP (GreenSock) for scroll-triggered timelines and complex UI interactions.
- **Fluid Motion:** Integrated **Lenis Smooth Scroll** for a premium, buttery-smooth browsing experience.
- **Interactive UX:** Custom-built "Image Trail" mouse interactions using a blend of React Hooks and GSAP.
- **Modern UI:** Designed with **Tailwind CSS v4** and Framer Motion for responsive, accessible, and beautiful components.
- **Performance Optimized:** Built on **Vite** for lightning-fast HMR and optimized production bundling.

---

## 🛠️ Tech Stack

### Frontend Core
- **React 19**: Utilizing the latest features for component-based UI.
- **React Router 7**: Managing seamless client-side navigation.
- **Tailwind CSS v4**: Utility-first styling with the latest engine performance.

### Animation & Motion
- **GSAP (GreenSock)**: The engine behind the complex scroll effects and timelines.
- **Framer Motion**: Used for micro-interactions and layout transitions.
- **Lenis**: Smooth scrolling management.
- **Swiper.js**: High-performance touch sliders and carousels.

### Tooling
- **Vite**: Next-generation frontend tooling.
- **Lucide & React Icons**: Crisp, scalable iconography.
- **ESLint**: Standardized code quality and linting.
